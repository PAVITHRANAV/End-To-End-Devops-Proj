# End-to-End DevOps CI/CD Pipeline

## 📌 Project Overview

A complete CI/CD pipeline that automatically builds, pushes, and deploys a web application using Jenkins, Docker, Docker Hub, and Kubernetes.

## 🚀 Technologies Used

- Git
- GitHub
- GitHub Webhooks
- Jenkins
- Docker
- Docker Hub
- Kubernetes
- Nginx

## ⚙️ Pipeline Flow

Developer Push

↓

GitHub

↓

Webhook

↓

Jenkins

↓

Docker Build

↓

Docker Hub

↓

Kubernetes Deployment

↓

Website Live
